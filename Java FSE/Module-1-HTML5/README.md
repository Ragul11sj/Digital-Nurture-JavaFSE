# CivicPulse - Local Community Event Portal
A modern and responsive community event portal built using HTML5, CSS3, and JavaScript. The platform helps users discover local events, explore community activities, register for events, save preferences, and locate nearby programs using browser geolocation.
## Project Overview
CivicPulse is a front-end web application designed to connect local communities through cultural, wellness, technology, food, music, and art events. The project focuses on responsive design, user engagement, accessibility, and modern UI/UX principles.
## Features
* Responsive navigation menu
* Attractive hero banner section
* Upcoming events showcase
* Event gallery with images
* Event registration form
* Event fee and schedule tables
* Embedded promotional video
* User preference management using Local Storage
* Geolocation-based event discovery
* Contact information section
* Mobile-friendly responsive design
* Modern typography and premium styling
## Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Geolocation API
* Google Fonts

## Functionalities

### Event Registration

Users can submit registration details including:

* Full Name
* Email Address
* Preferred Event Date
* Event Type
* Additional Message

### Preference Management

Users can:

* Save favorite event categories
* Store preferences in browser local storage
* Retrieve saved preferences automatically
* Clear preferences anytime

### Geolocation Services

The application can:

* Access user location
* Display coordinates
* Help users find nearby events
